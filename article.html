What is Git..?

Git - version control system

Git is a ten source distributed version-control system for tracking changes in source code during development. It is designed for coordinating work among programmers, but it can be used to track changes in any set of files. It was created by Linus Torvalds in 2005 to develop Linux Kernel. In simple Words it is used to manage or record the history of our source code
